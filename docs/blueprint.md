# **App Name**: PrepWise

## Core Features:

- Personalized Dashboard: Dashboard: An intuitive display showing upcoming exams, quick access to recent tests, and a summary of performance.
- Test UI: Test Taking: Present questions, multiple choice selections, and a timer.
- Answer Review: Answer Review: Highlight the selected and correct answers
- Visual Analytics: Performance Charts: Use line charts to show score progression on each subject. Display chapter proficiency via bar charts.
- Adaptive Recommendation: AI-Powered Insights Tool: The AI considers past performances to highlight which chapters to focus on to improve your overall test score.
- Performance History: Practice History: Store a comprehensive history of attempted tests and their corresponding results, allowing users to revisit past assessments and track their development over time.
- About / Contact / FAQ: Informational Pages: Include an 'About Us' section to build trust and provide background, a 'Contact Us' page for support and inquiries, and an 'FAQ' section to address common questions and concerns.

## Style Guidelines:

- Primary color: A deep, thoughtful blue (#3F51B5) evoking a sense of calm focus, facilitating concentration.
- Background color: A very light blue (#E8EAF6), almost white, to ensure maximum legibility and reduce eye strain during long study sessions. This keeps the interface light and airy.
- Accent color: A subdued violet (#7E57C2), used sparingly to highlight key interactive elements. This adds a touch of sophistication without overwhelming the user.
- Clean, sans-serif typography to ensure readability.
- A clear, sectioned layout to help focus study.
- Consistent iconography to improve the user experience.
